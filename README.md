## 1. Text — Классификация токенов (Hugging Face)

### **Задача:**  
Необходимо определить тип именованных сущностей в тексте (имена, организации, географические объекты и т. д.).

**Входные данные:** 
```python
Arkady Volozh founded Yandex in Moscow, Russia, in 1997, building it into one of the leading technology companies in Eastern Europe.
```
**Выходные данные:**  
```python
Ark - B-PER - 0.9976
##ady - B-PER - 0.8354
Vol - I-PER - 0.9972
##oz - I-PER - 0.9962
##h - I-PER - 0.9677
Yan - B-ORG - 0.9980
##de - I-ORG - 0.9927
##x - I-ORG - 0.9972
Moscow - B-LOC - 0.9997
Russia - B-LOC - 0.9998
Eastern - B-LOC - 0.9980
Europe - I-LOC - 0.9959
```
### **Метки классов:**

| Метка | Описание |
|:------:|-----------|
| **O** | Вне именованной сущности |
| **B-MISC** | Начало сущности (прочее) |
| **I-MISC** | Продолжение сущности (прочее) |
| **B-PER** | Начало имени человека |
| **I-PER** | Продолжение имени человека |
| **B-ORG** | Начало названия организации |
| **I-ORG** | Продолжение названия организации |
| **B-LOC** | Начало географического объекта |
| **I-LOC** | Продолжение географического объекта |

### **Реализация:**  
`\text.py`
